Installation
============
Manual
------
`curl https://raw.githubusercontent.com/extsoft/elegant-git/master/install.bash | $(which bash)`

For development
---------------
1. clone the repository
2. run `./install.bash dev`

Usage
=====
```bash
$ git elegant 
feature
pull
push
push-after-rebase
rebase
init
clone
add
clear-local
```
